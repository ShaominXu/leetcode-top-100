# 1. 两数之和

- LeetCode：[两数之和](https://leetcode.cn/problems/two-sum/)
- 分类：哈希表 / 数组

## 题意

给定整数数组 `nums` 和整数 `target`，在数组中找出两个数使得它们之和等于 `target`，返回这两个数的**下标**。

- 每个输入只有唯一解
- 不能使用同一个元素两次
- 返回下标的顺序不重要

## 思路

### 哈希表（推荐）

用哈希表 (字典) 记录已遍历的元素及其下标。遍历数组，对于每个元素 `num`，计算 `complement = target - num`：
- 如果 `complement` 已在哈希表中，找到答案
- 否则，将当前元素加入哈希表

**时间复杂度**：$O(n)$  
**空间复杂度**：$O(n)$

### 暴力法（不推荐）

双层循环，找到符合条件的两个数。  
**时间复杂度**：$O(n^2)$

## 代码

=== "Python"
    ```python
    from typing import List


    class Solution:
        def twoSum(self, nums: List[int], target: int) -> List[int]:
            # 记录 {值: 下标}
            num_map = {}
            for i, num in enumerate(nums):
                complement = target - num
                if complement in num_map:
                    return [num_map[complement], i]
                num_map[num] = i
            return []
    ```

=== "C++"
    ```cpp
    #include <vector>
    #include <unordered_map>
    using namespace std;


    class Solution {
    public:
        vector<int> twoSum(vector<int>& nums, int target) {
            unordered_map<int, int> num_map;
            for (int i = 0; i < (int)nums.size(); ++i) {
                int complement = target - nums[i];
                if (num_map.find(complement) != num_map.end()) {
                    return {num_map[complement], i};
                }
                num_map[nums[i]] = i;
            }
            return {};
        }
    };
    ```

## 示例

输入：`nums = [2, 7, 11, 15]`, `target = 9`

遍历过程：
- `i=0, num=2`: `complement=7` 不在哈希表中，添加 `{2: 0}`
- `i=1, num=7`: `complement=2` 在哈希表中（下标为 0），返回 `[0, 1]`

输出：`[0, 1]`

## 时间空间权衡

| 方法 | 时间 | 空间 |
|------|------|------|
| 暴力双层循环 | $O(n^2)$ | $O(1)$ |
| **哈希表** | $O(n)$ | $O(n)$ |

对于大规模数据，哈希表是最优选择。
