# [最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/)

## 思路

用哈希集合快速判断元素是否存在。若 `x-1` 不在集合中，说明 `x` 是某段连续序列的起点，
从 `x` 向后不断检查 `x+1, x+2...`，统计长度并更新答案。
每个数最多被访问常数次。

## 代码

=== "Python3"
    ```python

    class Solution:
        def longestConsecutive(self, nums: List[int]) -> int:
            st = set(nums)
            ans = 0
            for x in st:
                if x - 1 in st:
                    continue
                y = x
                while y in st:
                    y += 1
                ans = max(ans, y - x)
            return ans
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int longestConsecutive(vector<int>& nums) {
            unordered_set<int> st(nums.begin(), nums.end());
            int ans = 0;
            for (int x : st) {
                if (st.count(x - 1)) continue;
                int y = x;
                while (st.count(y)) ++y;
                ans = max(ans, y - x);
            }
            return ans;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(n)，额外空间主要来自哈希表/队列/递归栈等辅助结构。
