# [只出现一次的数字](https://leetcode.cn/problems/single-number/)

## 思路

利用异或性质：

- a \oplus a = 0
- a \oplus 0 = a
- 异或满足交换律和结合律

把所有数字异或起来，成对出现的数会抵消，剩下只出现一次的数字。

## 代码

=== "Python3"
    ```python

    class Solution:
        def singleNumber(self, nums: List[int]) -> int:
            x = 0
            for v in nums:
                x ^= v
            return x
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int singleNumber(vector<int>& nums) {
            int x = 0;
            for (int v : nums) x ^= v;
            return x;
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n)。
空间复杂度：O(1)。
