# [乘积最大子数组](https://leetcode.cn/problems/maximum-product-subarray/)

## 思路

因为负数会让最大最小互换，遍历时同时维护以当前位置结尾的最大乘积 `mx` 和最小乘积 `mn`。
遇到负数先交换 `mx`、`mn`，再更新。

## 代码

=== "Python3"
    ```python

    class Solution:
        def maxProduct(self, nums: List[int]) -> int:
            mx = mn = ans = nums[0]
            for x in nums[1:]:
                if x < 0:
                    mx, mn = mn, mx
                mx = max(x, mx * x)
                mn = min(x, mn * x)
                ans = max(ans, mx)
            return ans
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int maxProduct(vector<int>& nums) {
            int mx = nums[0], mn = nums[0], ans = nums[0];
            for (int i = 1; i < (int)nums.size(); ++i) {
                int x = nums[i];
                if (x < 0) swap(mx, mn);
                mx = max(x, mx * x);
                mn = min(x, mn * x);
                ans = max(ans, mx);
            }
            return ans;
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n)。
空间复杂度：O(1)。
