# [155. 最小栈](https://leetcode.cn/problems/min-stack/)

## 思路

维护两个栈：
- `st` 存正常值
- `mn` 存到当前位置的最小值
每次入栈时同时更新 `mn`，出栈时两个栈一起弹出。

时间复杂度：各操作均为 $O(1)$。
空间复杂度：$O(n)$。

## 代码

=== "Python"
    ```python
    class MinStack:
        def __init__(self):
            self.st = []
            self.mn = []

        def push(self, val: int) -> None:
            self.st.append(val)
            if not self.mn:
                self.mn.append(val)
            else:
                self.mn.append(min(val, self.mn[-1]))

        def pop(self) -> None:
            self.st.pop()
            self.mn.pop()

        def top(self) -> int:
            return self.st[-1]

        def getMin(self) -> int:
            return self.mn[-1]
    ```

=== "C++"
    ```cpp
    #include <vector>
    #include <algorithm>
    using namespace std;

    class MinStack {
    public:
        vector<int> st, mn;

        MinStack() {}

        void push(int val) {
            st.push_back(val);
            if (mn.empty()) mn.push_back(val);
            else mn.push_back(min(val, mn.back()));
        }

        void pop() {
            st.pop_back();
            mn.pop_back();
        }

        int top() {
            return st.back();
        }

        int getMin() {
            return mn.back();
        }
    };
    ```
