# [相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/)

## 思路

双指针 `a`、`b` 分别从两链表头出发，走到末尾后切换到另一条链表头。
两指针总路程相同，若有相交会在交点相遇，否则同时为 `None`。

时间复杂度：O(m+n)。
空间复杂度：O(1)。

## 代码

=== "Python"
    ```python
    class ListNode:
        def __init__(self, x):
            self.val = x
            self.next = None

    class Solution:
        def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> ListNode:
            a, b = headA, headB
            while a is not b:
                a = a.next if a else headB
                b = b.next if b else headA
            return a
    ```

=== "C++"
    ```cpp
    using namespace std;

    struct ListNode {
        int val;
        ListNode* next;
        ListNode(int x) : val(x), next(nullptr) {}
    };

    class Solution {
    public:
        ListNode* getIntersectionNode(ListNode* headA, ListNode* headB) {
            ListNode* a = headA;
            ListNode* b = headB;
            while (a != b) {
                a = a ? a->next : headB;
                b = b ? b->next : headA;
            }
            return a;
        }
    };
    ```
