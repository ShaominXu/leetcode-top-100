# [17. 电话号码的字母组合](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)

## 思路

回溯。按数字顺序逐位选择一个字母加入路径，直到路径长度等于数字串长度。
空字符串直接返回空数组。

时间复杂度：$O(4^n \cdot n)$。
空间复杂度：$O(n)$（不计输出）。

## 代码

=== "Python"
    ```python
    from typing import List

    class Solution:
        def letterCombinations(self, digits: str) -> List[str]:
            if not digits:
                return []
            mp = {
                '2': 'abc', '3': 'def', '4': 'ghi', '5': 'jkl',
                '6': 'mno', '7': 'pqrs', '8': 'tuv', '9': 'wxyz'
            }
            ans = []
            path = []

            def dfs(i: int):
                if i == len(digits):
                    ans.append(''.join(path))
                    return
                for ch in mp[digits[i]]:
                    path.append(ch)
                    dfs(i + 1)
                    path.pop()

            dfs(0)
            return ans
    ```

=== "C++"
    ```cpp
    #include <vector>
    #include <string>
    #include <unordered_map>
    using namespace std;

    class Solution {
    public:
        vector<string> ans;
        string path;
        unordered_map<char, string> mp = {
            {'2', "abc"}, {'3', "def"}, {'4', "ghi"}, {'5', "jkl"},
            {'6', "mno"}, {'7', "pqrs"}, {'8', "tuv"}, {'9', "wxyz"}
        };

        void dfs(const string& digits, int i) {
            if (i == (int)digits.size()) {
                ans.push_back(path);
                return;
            }
            for (char ch : mp[digits[i]]) {
                path.push_back(ch);
                dfs(digits, i + 1);
                path.pop_back();
            }
        }

        vector<string> letterCombinations(string digits) {
            if (digits.empty()) return {};
            dfs(digits, 0);
            return ans;
        }
    };
    ```
