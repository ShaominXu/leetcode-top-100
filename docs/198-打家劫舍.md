# [打家劫舍](https://leetcode.cn/problems/house-robber/)

## 思路

线性 DP。遍历每个房屋金额 `x`：

- `take` 表示抢当前房：`prev_not_take + x`
- `not_take` 表示不抢当前房：`max(prev_take, prev_not_take)`
可滚动为两个变量。

## 代码

=== "Python3"
    ```python

    class Solution:
        def rob(self, nums: List[int]) -> int:
            take = 0
            not_take = 0
            for x in nums:
                take, not_take = not_take + x, max(take, not_take)
            return max(take, not_take)
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int rob(vector<int>& nums) {
            int take = 0, not_take = 0;
            for (int x : nums) {
                int new_take = not_take + x;
                int new_not_take = max(take, not_take);
                take = new_take;
                not_take = new_not_take;
            }
            return max(take, not_take);
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n)。
空间复杂度：O(1)。
