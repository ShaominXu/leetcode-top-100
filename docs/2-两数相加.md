# [两数相加](https://leetcode.cn/problems/add-two-numbers/)

## 思路

从低位到高位同时遍历两条链表，维护进位 `carry`。
每步计算 `sum = x + y + carry`，当前位是 `sum % 10`，新的进位是 `sum // 10`。
直到两条链表都结束且进位为 0。


## 代码

=== "Python3"
    ```python
    class ListNode:
        def __init__(self, val=0, next=None):
            self.val = val
            self.next = next

    class Solution:
        def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
            dummy = ListNode()
            cur = dummy
            carry = 0
            while l1 or l2 or carry:
                x = l1.val if l1 else 0
                y = l2.val if l2 else 0
                s = x + y + carry
                carry = s // 10
                cur.next = ListNode(s % 10)
                cur = cur.next
                l1 = l1.next if l1 else None
                l2 = l2.next if l2 else None
            return dummy.next
    ```

=== "C++"
    ```cpp
    struct ListNode {
        int val;
        ListNode* next;
        ListNode(int x = 0, ListNode* n = nullptr) : val(x), next(n) {}
    };

    class Solution {
    public:
        ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
            ListNode dummy;
            ListNode* cur = &dummy;
            int carry = 0;
            while (l1 || l2 || carry) {
                int x = l1 ? l1->val : 0;
                int y = l2 ? l2->val : 0;
                int s = x + y + carry;
                carry = s / 10;
                cur->next = new ListNode(s % 10);
                cur = cur->next;
                l1 = l1 ? l1->next : nullptr;
                l2 = l2 ? l2->next : nullptr;
            }
            return dummy.next;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(max(m,n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(max(m,n)，额外空间主要来自哈希表/队列/递归栈等辅助结构。
