# [20. 有效的括号](https://leetcode.cn/problems/valid-parentheses/)

## 思路

用栈匹配括号。遇到左括号入栈，遇到右括号时检查栈顶是否是对应左括号。
若不匹配或栈空则返回 false，最终栈为空则合法。

时间复杂度：$O(n)$。
空间复杂度：$O(n)$。

## 代码

=== "Python"
    ```python
    class Solution:
        def isValid(self, s: str) -> bool:
            mp = {')': '(', ']': '[', '}': '{'}
            st = []
            for ch in s:
                if ch in '([{':
                    st.append(ch)
                else:
                    if not st or st[-1] != mp[ch]:
                        return False
                    st.pop()
            return not st
    ```

=== "C++"
    ```cpp
    #include <string>
    #include <stack>
    using namespace std;

    class Solution {
    public:
        bool isValid(string s) {
            stack<char> st;
            for (char ch : s) {
                if (ch == '(' || ch == '[' || ch == '{') {
                    st.push(ch);
                } else {
                    if (st.empty()) return false;
                    char t = st.top();
                    st.pop();
                    if ((ch == ')' && t != '(') || (ch == ']' && t != '[') || (ch == '}' && t != '{')) {
                        return false;
                    }
                }
            }
            return st.empty();
        }
    };
    ```
