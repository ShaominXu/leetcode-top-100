# [数组中的第 K 个最大元素](https://leetcode.cn/problems/kth-largest-element-in-an-array/)

## 思路

维护大小为 `k` 的最小堆。
遍历数组时先入堆，若堆大小超过 `k` 则弹出堆顶。遍历结束后堆顶就是第 `k` 大元素。

## 代码

=== "Python3"
    ```python
    import heapq

    class Solution:
        def findKthLargest(self, nums: List[int], k: int) -> int:
            hp = []
            for x in nums:
                heapq.heappush(hp, x)
                if len(hp) > k:
                    heapq.heappop(hp)
            return hp[0]
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int findKthLargest(vector<int>& nums, int k) {
            priority_queue<int, vector<int>, greater<int>> pq;
            for (int x : nums) {
                pq.push(x);
                if ((int)pq.size() > k) pq.pop();
            }
            return pq.top();
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n\log k)。
空间复杂度：O(k)。
