# [23. 合并 K 个升序链表](https://leetcode.cn/problems/merge-k-sorted-lists/)

## 思路

使用最小堆维护每条链表当前结点。每次弹出最小结点接到结果链表后，
若该结点有后继，则将后继入堆。

时间复杂度：$O(N\log k)$，$N$ 为总节点数。
空间复杂度：$O(k)$。

## 代码

=== "Python"
    ```python
    import heapq

    class ListNode:
        def __init__(self, val=0, next=None):
            self.val = val
            self.next = next

    class Solution:
        def mergeKLists(self, lists):
            hp = []
            for i, node in enumerate(lists):
                if node:
                    heapq.heappush(hp, (node.val, i, node))
            dummy = ListNode()
            cur = dummy
            while hp:
                _, i, node = heapq.heappop(hp)
                cur.next = node
                cur = cur.next
                if node.next:
                    heapq.heappush(hp, (node.next.val, i, node.next))
            return dummy.next
    ```

=== "C++"
    ```cpp
    #include <vector>
    #include <queue>
    using namespace std;

    struct ListNode {
        int val;
        ListNode* next;
        ListNode(int x = 0, ListNode* n = nullptr) : val(x), next(n) {}
    };

    class Solution {
    public:
        struct Cmp {
            bool operator()(ListNode* a, ListNode* b) const {
                return a->val > b->val;
            }
        };

        ListNode* mergeKLists(vector<ListNode*>& lists) {
            priority_queue<ListNode*, vector<ListNode*>, Cmp> pq;
            for (auto* node : lists) {
                if (node) pq.push(node);
            }
            ListNode dummy;
            ListNode* cur = &dummy;
            while (!pq.empty()) {
                ListNode* node = pq.top();
                pq.pop();
                cur->next = node;
                cur = cur->next;
                if (node->next) pq.push(node->next);
            }
            return dummy.next;
        }
    };
    ```
