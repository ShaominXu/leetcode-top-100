# [K 个一组翻转链表](https://leetcode.cn/problems/reverse-nodes-in-k-group/)

## 思路

每次检查从当前起点开始是否有 `k` 个结点。
若不足 `k` 个直接结束；若足够则原地翻转这一组，并把前后链表接起来。

时间复杂度：O(n)。
空间复杂度：O(1)。

## 代码

=== "Python"
    ```python
    class ListNode:
        def __init__(self, val=0, next=None):
            self.val = val
            self.next = next

    class Solution:
        def reverseKGroup(self, head: ListNode, k: int) -> ListNode:
            dummy = ListNode(0, head)
            pre = dummy
            while True:
                end = pre
                for _ in range(k):
                    end = end.next
                    if not end:
                        return dummy.next
                start = pre.next
                nxt = end.next
                prev, cur = nxt, start
                while cur != nxt:
                    tmp = cur.next
                    cur.next = prev
                    prev = cur
                    cur = tmp
                pre.next = end
                pre = start
    ```

=== "C++"
    ```cpp
    using namespace std;

    struct ListNode {
        int val;
        ListNode* next;
        ListNode(int x = 0, ListNode* n = nullptr) : val(x), next(n) {}
    };

    class Solution {
    public:
        ListNode* reverseKGroup(ListNode* head, int k) {
            ListNode dummy(0, head);
            ListNode* pre = &dummy;
            while (true) {
                ListNode* end = pre;
                for (int i = 0; i < k; ++i) {
                    end = end->next;
                    if (!end) return dummy.next;
                }
                ListNode* start = pre->next;
                ListNode* nxt = end->next;
                ListNode* prev = nxt;
                ListNode* cur = start;
                while (cur != nxt) {
                    ListNode* tmp = cur->next;
                    cur->next = prev;
                    prev = cur;
                    cur = tmp;
                }
                pre->next = end;
                pre = start;
            }
        }
    };
    ```
