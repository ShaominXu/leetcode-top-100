# [完全平方数](https://leetcode.cn/problems/perfect-squares/)

## 思路

完全背包。`dp[i]` 表示和为 `i` 的最少完全平方数数量。
枚举平方数 `s*j=s`，更新 `dp[i] = min(dp[i], dp[i-s] + 1)`。

## 代码

=== "Python3"
    ```python
    class Solution:
        def numSquares(self, n: int) -> int:
            dp = [10**9] * (n + 1)
            dp[0] = 0
            s = 1
            while s * s <= n:
                sq = s * s
                for i in range(sq, n + 1):
                    dp[i] = min(dp[i], dp[i - sq] + 1)
                s += 1
            return dp[n]
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int numSquares(int n) {
            vector<int> dp(n + 1, 1e9);
            dp[0] = 0;
            for (int s = 1; s * s <= n; ++s) {
                int sq = s * s;
                for (int i = sq; i <= n; ++i) {
                    dp[i] = min(dp[i], dp[i - sq] + 1);
                }
            }
            return dp[n];
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n sqrt(n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(n)，额外空间主要来自哈希表/队列/递归栈等辅助结构。
