# [移动零](https://leetcode.cn/problems/move-zeroes/)

## 思路

双指针。`k` 指向下一个应放非零元素的位置。
遍历数组，把每个非零元素交换到 `k`，然后 `k += 1`。
最终所有非零元素保持相对顺序，零被移到末尾。

## 代码

=== "Python3"
    ```python

    class Solution:
        def moveZeroes(self, nums: List[int]) -> None:
            k = 0
            for i in range(len(nums)):
                if nums[i] != 0:
                    nums[k], nums[i] = nums[i], nums[k]
                    k += 1
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        void moveZeroes(vector<int>& nums) {
            int k = 0;
            for (int i = 0; i < (int)nums.size(); ++i) {
                if (nums[i] != 0) {
                    swap(nums[k], nums[i]);
                    ++k;
                }
            }
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(1)，因为只使用常数级额外变量。
