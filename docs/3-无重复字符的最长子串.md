# [3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

## 思路

使用滑动窗口。维护左边界 `left` 和哈希表 `last`（记录字符最近出现位置）。
遍历到下标 `right` 时，如果当前字符在窗口内出现过，则把 `left` 更新到重复字符上次位置的下一位。
窗口长度为 `right - left + 1`，遍历过程中维护最大值。

时间复杂度：$O(n)$。
空间复杂度：$O(\min(n, |\Sigma|))$。

## 代码

=== "Python"
    ```python
    class Solution:
        def lengthOfLongestSubstring(self, s: str) -> int:
            last = {}
            left = 0
            ans = 0
            for right, ch in enumerate(s):
                if ch in last and last[ch] >= left:
                    left = last[ch] + 1
                last[ch] = right
                ans = max(ans, right - left + 1)
            return ans
    ```

=== "C++"
    ```cpp
    #include <string>
    #include <unordered_map>
    #include <algorithm>
    using namespace std;

    class Solution {
    public:
        int lengthOfLongestSubstring(string s) {
            unordered_map<char, int> last;
            int left = 0, ans = 0;
            for (int right = 0; right < (int)s.size(); ++right) {
                char ch = s[right];
                if (last.count(ch) && last[ch] >= left) {
                    left = last[ch] + 1;
                }
                last[ch] = right;
                ans = max(ans, right - left + 1);
            }
            return ans;
        }
    };
    ```
