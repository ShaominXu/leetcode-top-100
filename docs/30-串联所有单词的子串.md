# 30. 串联所有单词的子串

给定一个字符串 `s` 和一个字符串数组 `words`，`words` 中所有字符串长度相同。`s` 中的 **串联子串** 是指一个包含 `words` 中所有字符串恰好一次排列连接而成的子串。

例如，如果 `words = ["ab","cd","ef"]`，那么 `"abcdef"` 是一个串联子串，因为它由 `"ab"` + `"cd"` + `"ef"` 组成。返回 `s` 中所有串联子串的起始索引。您可以以任意顺序返回答案。

**示例：**
- 输入：s = "barfoothefoobarman", words = ["foo","bar"]
- 输出：[0,9]

解释：串联子串从索引 0 和 9 开始。

## 解题思路

使用滑动窗口结合哈希表：

1. **建立单词计数**：先用哈希表统计 `words` 中每个单词的出现次数
2. **滑动窗口**：在 `s` 中维护一个固定大小的窗口（总长度 = words长度 × 每个单词长度）
3. **逐步扫描**：从每个可能的起始位置尝试，检查是否能凑出所有单词
4. **记录结果**：每次匹配成功就记录起始索引

## 核心代码

```python
def findSubstring(s: str, words: List[str]) -> List[int]:
    if not s or not words:
        return []
    
    word_len = len(words[0])
    word_count = len(words)
    total_len = word_len * word_count
    word_map = {}
    
    # 统计每个单词的出现次数
    for word in words:
        word_map[word] = word_map.get(word, 0) + 1
    
    result =[]
    
    # 从每个起始位置检查
    for i in range(len(s) - total_len + 1):
        # 检查该位置后的 total_len 长度内是否能凑出所有单词
        window_map = {}
        for j in range(i, i + total_len, word_len):
            word = s[j:j + word_len]
            if word not in word_map:
                break
            window_map[word] = window_map.get(word, 0) + 1
            if window_map[word] > word_map[word]:
                break
        else:
            # 检查是否完全匹配
            if window_map == word_map:
                result.append(i)
    
    return result
```

## 复杂度分析

**时间复杂度**：**O(n × m)**，因为需要扫描字符串中的每个可能位置，每个位置检查需要提取单词并比较。

**空间复杂度**：**O(m)**，因为哈希表存储单词及其计数。

## 关键点

- ✅ 利用滑动窗口固定大小的特点
- ✅ 单词长度相同简化了窗口管理
- ✅ 哈希表快速查找和计数
- ✅ 避免重复检查相同子串

## 变体与扩展

- 单词长度不同时的处理
- 包含重复单词的情况
- 优化到 O(n) 时间的方法
