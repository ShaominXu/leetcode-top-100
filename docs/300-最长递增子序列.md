# [最长递增子序列](https://leetcode.cn/problems/longest-increasing-subsequence/)

## 思路

使用贪心 + 二分。维护数组 `tails`：`tails[i]` 表示长度为 `i+1` 的递增子序列最小结尾值。
遍历每个数 `x`，在 `tails` 中二分找到第一个大于等于 `x` 的位置替换；若不存在则追加。
`tails` 长度即答案。

## 代码

=== "Python3"
    ```python
    import bisect

    class Solution:
        def lengthOfLIS(self, nums: List[int]) -> int:
            tails = []
            for x in nums:
                i = bisect.bisect_left(tails, x)
                if i == len(tails):
                    tails.append(x)
                else:
                    tails[i] = x
            return len(tails)
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int lengthOfLIS(vector<int>& nums) {
            vector<int> tails;
            for (int x : nums) {
                auto it = lower_bound(tails.begin(), tails.end(), x);
                if (it == tails.end()) tails.push_back(x);
                else *it = x;
            }
            return (int)tails.size();
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n \log n)。
空间复杂度：O(n)。
