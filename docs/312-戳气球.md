# [戳气球](https://leetcode.cn/problems/burst-balloons/)

## 思路

区间 DP。两端补 1，设 `dp[l][r]` 表示开区间 `(l,r)` 全部戳完的最大硬币数。
枚举最后一个戳的气球 `k`：
`dp[l][r] = max(dp[l][k] + dp[k][r] + val[l]*val[k]*val[r])`。

## 代码

=== "Python3"
    ```python

    class Solution:
        def maxCoins(self, nums: List[int]) -> int:
            val = [1] + nums + [1]
            n = len(val)
            dp = [[0] * n for _ in range(n)]
            for length in range(2, n):
                for l in range(0, n - length):
                    r = l + length
                    for k in range(l + 1, r):
                        dp[l][r] = max(dp[l][r], dp[l][k] + dp[k][r] + val[l] * val[k] * val[r])
            return dp[0][n - 1]
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int maxCoins(vector<int>& nums) {
            vector<int> val;
            val.push_back(1);
            for (int x : nums) val.push_back(x);
            val.push_back(1);
            int n = (int)val.size();
            vector<vector<int>> dp(n, vector<int>(n, 0));
            for (int len = 2; len < n; ++len) {
                for (int l = 0; l + len < n; ++l) {
                    int r = l + len;
                    for (int k = l + 1; k < r; ++k) {
                        dp[l][r] = max(dp[l][r], dp[l][k] + dp[k][r] + val[l] * val[k] * val[r]);
                    }
                }
            }
            return dp[0][n - 1];
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n³)。
空间复杂度：O(n²)。
