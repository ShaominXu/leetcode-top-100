# [32. 最长有效括号](https://leetcode.cn/problems/longest-valid-parentheses/)

## 思路

栈中存下标，初始压入 `-1` 作为哨兵。
- 遇到 `(` 入栈下标
- 遇到 `)` 弹栈，若栈空则压入当前下标作为新起点
- 否则用 `i - st.top()` 更新最长长度

时间复杂度：$O(n)$。
空间复杂度：$O(n)$。

## 代码

=== "Python"
    ```python
    class Solution:
        def longestValidParentheses(self, s: str) -> int:
            st = [-1]
            ans = 0
            for i, ch in enumerate(s):
                if ch == '(':
                    st.append(i)
                else:
                    st.pop()
                    if not st:
                        st.append(i)
                    else:
                        ans = max(ans, i - st[-1])
            return ans
    ```

=== "C++"
    ```cpp
    #include <string>
    #include <stack>
    #include <algorithm>
    using namespace std;

    class Solution {
    public:
        int longestValidParentheses(string s) {
            stack<int> st;
            st.push(-1);
            int ans = 0;
            for (int i = 0; i < (int)s.size(); ++i) {
                if (s[i] == '(') {
                    st.push(i);
                } else {
                    st.pop();
                    if (st.empty()) st.push(i);
                    else ans = max(ans, i - st.top());
                }
            }
            return ans;
        }
    };
    ```
