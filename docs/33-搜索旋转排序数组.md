# [搜索旋转排序数组](https://leetcode.cn/problems/search-in-rotated-sorted-array/)

## 思路

二分时至少有一半区间有序。
比较 `nums[mid]` 与边界值，判断哪一半有序，再判断目标值是否落在有序半区内，据此缩小范围。

## 代码

=== "Python3"
    ```python

    class Solution:
        def search(self, nums: List[int], target: int) -> int:
            l, r = 0, len(nums) - 1
            while l <= r:
                m = (l + r) // 2
                if nums[m] == target:
                    return m
                if nums[l] <= nums[m]:
                    if nums[l] <= target < nums[m]:
                        r = m - 1
                    else:
                        l = m + 1
                else:
                    if nums[m] < target <= nums[r]:
                        l = m + 1
                    else:
                        r = m - 1
            return -1
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int search(vector<int>& nums, int target) {
            int l = 0, r = (int)nums.size() - 1;
            while (l <= r) {
                int m = l + (r - l) / 2;
                if (nums[m] == target) return m;
                if (nums[l] <= nums[m]) {
                    if (nums[l] <= target && target < nums[m]) r = m - 1;
                    else l = m + 1;
                } else {
                    if (nums[m] < target && target <= nums[r]) l = m + 1;
                    else r = m - 1;
                }
            }
            return -1;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(log n)，主要开销来自排序/二分/堆等对数级操作。
空间复杂度：O(1)，因为只使用常数级额外变量。
