# [打家劫舍 III](https://leetcode.cn/problems/house-robber-iii/)

## 思路

树形 DP。对每个结点返回两个值：

- `rob`：抢当前结点的最大收益
- `not_rob`：不抢当前结点的最大收益
转移：

- `rob = node.val + left.not_rob + right.not_rob`
- `not_rob = max(left.rob,left.not_rob) + max(right.rob,right.not_rob)`

## 代码

=== "Python3"
    ```python
    from typing import Optional, Tuple

    class TreeNode:
        def __init__(self, val=0, left=None, right=None):
            self.val = val
            self.left = left
            self.right = right

    class Solution:
        def rob(self, root: Optional[TreeNode]) -> int:
            def dfs(node: Optional[TreeNode]) -> Tuple[int, int]:
                if not node:
                    return 0, 0
                lr, ln = dfs(node.left)
                rr, rn = dfs(node.right)
                rob_cur = node.val + ln + rn
                not_rob_cur = max(lr, ln) + max(rr, rn)
                return rob_cur, not_rob_cur

            r, n = dfs(root)
            return max(r, n)
    ```

=== "C++"
    ```cpp

    struct TreeNode {
        int val;
        TreeNode* left;
        TreeNode* right;
        TreeNode(int x = 0, TreeNode* l = nullptr, TreeNode* r = nullptr) : val(x), left(l), right(r) {}
    };

    class Solution {
    public:
        pair<int, int> dfs(TreeNode* node) {
            if (!node) return {0, 0};
            auto [lr, ln] = dfs(node->left);
            auto [rr, rn] = dfs(node->right);
            int rob_cur = node->val + ln + rn;
            int not_rob_cur = max(lr, ln) + max(rr, rn);
            return {rob_cur, not_rob_cur};
        }

        int rob(TreeNode* root) {
            auto [r, n] = dfs(root);
            return max(r, n);
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n)。
空间复杂度：O(h)。
