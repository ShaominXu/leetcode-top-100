# [比特位计数](https://leetcode.cn/problems/counting-bits/)

## 思路

动态规划。利用最低位：
`dp[i] = dp[i >> 1] + (i & 1)`。
即右移一位去掉最低位，再加上最低位是否为 1。

## 代码

=== "Python3"
    ```python

    class Solution:
        def countBits(self, n: int) -> List[int]:
            dp = [0] * (n + 1)
            for i in range(1, n + 1):
                dp[i] = dp[i >> 1] + (i & 1)
            return dp
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        vector<int> countBits(int n) {
            vector<int> dp(n + 1, 0);
            for (int i = 1; i <= n; ++i) {
                dp[i] = dp[i >> 1] + (i & 1);
            }
            return dp;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(n)，额外空间主要来自哈希表/队列/递归栈等辅助结构。
