# [组合总和](https://leetcode.cn/problems/combination-sum/)

## 思路

回溯枚举每个候选数选几次。为避免重复组合，递归时传入起始下标 `start`，
下一层仍可从当前下标继续选，实现元素可重复使用。

## 代码

=== "Python3"
    ```python

    class Solution:
        def combinationSum(self, candidates: List[int], target: int) -> List[List[int]]:
            ans = []
            path = []

            def dfs(start: int, remain: int):
                if remain == 0:
                    ans.append(path[:])
                    return
                if remain < 0:
                    return
                for i in range(start, len(candidates)):
                    path.append(candidates[i])
                    dfs(i, remain - candidates[i])
                    path.pop()

            dfs(0, target)
            return ans
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        vector<vector<int>> ans;
        vector<int> path;

        void dfs(vector<int>& candidates, int start, int remain) {
            if (remain == 0) {
                ans.push_back(path);
                return;
            }
            if (remain < 0) return;
            for (int i = start; i < (int)candidates.size(); ++i) {
                path.push_back(candidates[i]);
                dfs(candidates, i, remain - candidates[i]);
                path.pop_back();
            }
        }

        vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
            dfs(candidates, 0, target);
            return ans;
        }
    };
    ```

### 复杂度分析

时间复杂度：与解空间规模相关，最坏可近似看作指数级。
空间复杂度：O(target)，额外空间主要来自哈希表/队列/递归栈等辅助结构。
