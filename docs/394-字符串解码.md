# [字符串解码](https://leetcode.cn/problems/decode-string/)

## 思路

使用栈处理嵌套结构。遍历字符串：

- 读到数字：累计当前重复次数
- 读到 `[`：把当前字符串和重复次数压栈，并重置
- 读到 `]`：弹栈并展开
- 读到字母：拼接到当前字符串

## 代码

=== "Python3"
    ```python
    class Solution:
        def decodeString(self, s: str) -> str:
            st = []
            cur_num = 0
            cur_str = ""
            for ch in s:
                if ch.isdigit():
                    cur_num = cur_num * 10 + int(ch)
                elif ch == '[':
                    st.append((cur_str, cur_num))
                    cur_str = ""
                    cur_num = 0
                elif ch == ']':
                    prev_str, k = st.pop()
                    cur_str = prev_str + cur_str * k
                else:
                    cur_str += ch
            return cur_str
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        string decodeString(string s) {
            vector<pair<string, int>> st;
            int cur_num = 0;
            string cur_str;
            for (char ch : s) {
                if (isdigit(ch)) {
                    cur_num = cur_num * 10 + (ch - '0');
                } else if (ch == '[') {
                    st.push_back({cur_str, cur_num});
                    cur_str.clear();
                    cur_num = 0;
                } else if (ch == ']') {
                    auto [prev_str, k] = st.back();
                    st.pop_back();
                    string t;
                    for (int i = 0; i < k; ++i) t += cur_str;
                    cur_str = prev_str + t;
                } else {
                    cur_str.push_back(ch);
                }
            }
            return cur_str;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(n)，额外空间主要来自哈希表/队列/递归栈等辅助结构。
