# [分割等和子集](https://leetcode.cn/problems/partition-equal-subset-sum/)

## 思路

总和为 `sum`，若 `sum` 为奇数则不可能。
目标转化为是否能选出若干数使和为 `sum/2`，是 0/1 背包可达性问题。
`dp[j]` 表示和 `j` 是否可达，倒序遍历保证每个元素只用一次。

时间复杂度：O(n \cdot sum)。
空间复杂度：O(sum)。

## 代码

=== "Python"
    ```python
    from typing import List

    class Solution:
        def canPartition(self, nums: List[int]) -> bool:
            s = sum(nums)
            if s % 2:
                return False
            target = s // 2
            dp = [False] * (target + 1)
            dp[0] = True
            for x in nums:
                for j in range(target, x - 1, -1):
                    dp[j] = dp[j] or dp[j - x]
            return dp[target]
    ```

=== "C++"
    ```cpp
    #include <vector>
    #include <numeric>
    using namespace std;

    class Solution {
    public:
        bool canPartition(vector<int>& nums) {
            int s = accumulate(nums.begin(), nums.end(), 0);
            if (s % 2) return false;
            int target = s / 2;
            vector<bool> dp(target + 1, false);
            dp[0] = true;
            for (int x : nums) {
                for (int j = target; j >= x; --j) {
                    dp[j] = dp[j] || dp[j - x];
                }
            }
            return dp[target];
        }
    };
    ```
