# [路径总和 III](https://leetcode.cn/problems/path-sum-iii/)

## 思路

前缀和 + DFS。
在根到当前结点路径上，若存在前缀和 `pre - target`，则对应一条满足条件的路径。
用哈希表记录当前路径各前缀和出现次数，递归进入子树前后分别加一和回溯减一。

## 代码

=== "Python3"
    ```python
    from typing import Optional
    from collections import defaultdict

    class TreeNode:
        def __init__(self, val=0, left=None, right=None):
            self.val = val
            self.left = left
            self.right = right

    class Solution:
        def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
            cnt = defaultdict(int)
            cnt[0] = 1

            def dfs(node: Optional[TreeNode], pre: int) -> int:
                if not node:
                    return 0
                pre += node.val
                ans = cnt[pre - targetSum]
                cnt[pre] += 1
                ans += dfs(node.left, pre)
                ans += dfs(node.right, pre)
                cnt[pre] -= 1
                return ans

            return dfs(root, 0)
    ```

=== "C++"
    ```cpp

    struct TreeNode {
        int val;
        TreeNode* left;
        TreeNode* right;
        TreeNode(int x = 0, TreeNode* l = nullptr, TreeNode* r = nullptr) : val(x), left(l), right(r) {}
    };

    class Solution {
    public:
        unordered_map<long long, int> cnt;
        int target;

        int dfs(TreeNode* node, long long pre) {
            if (!node) return 0;
            pre += node->val;
            int ans = cnt.count(pre - target) ? cnt[pre - target] : 0;
            ++cnt[pre];
            ans += dfs(node->left, pre);
            ans += dfs(node->right, pre);
            --cnt[pre];
            return ans;
        }

        int pathSum(TreeNode* root, int targetSum) {
            target = targetSum;
            cnt[0] = 1;
            return dfs(root, 0);
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n)。
空间复杂度：O(h) 到 O(n)。
