# [最长回文子串](https://leetcode.cn/problems/longest-palindromic-substring/)

## 思路

中心扩展。每个位置可作为奇数长度中心，也可与右侧组成偶数长度中心。
从中心向两侧扩展并更新最长区间。

时间复杂度：O(n^2)。
空间复杂度：O(1)。

## 代码

=== "Python"
    ```python
    class Solution:
        def longestPalindrome(self, s: str) -> str:
            n = len(s)
            if n < 2:
                return s
            start, max_len = 0, 1

            def expand(l: int, r: int) -> None:
                nonlocal start, max_len
                while l >= 0 and r < n and s[l] == s[r]:
                    l -= 1
                    r += 1
                l += 1
                r -= 1
                if r - l + 1 > max_len:
                    start = l
                    max_len = r - l + 1

            for i in range(n):
                expand(i, i)
                expand(i, i + 1)
            return s[start:start + max_len]
    ```

=== "C++"
    ```cpp
    #include <string>
    using namespace std;

    class Solution {
    public:
        string s;
        int n;
        int start = 0, max_len = 1;

        void expand(int l, int r) {
            while (l >= 0 && r < n && s[l] == s[r]) {
                --l;
                ++r;
            }
            ++l;
            --r;
            if (r - l + 1 > max_len) {
                start = l;
                max_len = r - l + 1;
            }
        }

        string longestPalindrome(string str) {
            s = str;
            n = (int)s.size();
            if (n < 2) return s;
            for (int i = 0; i < n; ++i) {
                expand(i, i);
                expand(i, i + 1);
            }
            return s.substr(start, max_len);
        }
    };
    ```
