# [最大子数组和](https://leetcode.cn/problems/maximum-subarray/)

## 思路

- 设 `cur` 为「以当前元素结尾」的子数组的最大和。
- 遍历每个 `x`：要么把 `x` 接在上一段后面（`cur + x`），要么从 `x` 重新开始一段（`x`），取较大者：  
  `cur = max(x, cur + x)`。
- 全局答案 `ans` 为所有位置 `cur` 的最大值。

## 代码

=== "Python3"
    ```python

    class Solution:
        def maxSubArray(self, nums: List[int]) -> int:
            cur = ans = nums[0]
            for x in nums[1:]:
                cur = max(x, cur + x)
                ans = max(ans, cur)
            return ans
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int maxSubArray(vector<int>& nums) {
            int cur = nums[0], ans = nums[0];
            for (int i = 1; i < (int)nums.size(); ++i) {
                cur = max(nums[i], cur + nums[i]);
                ans = max(ans, cur);
            }
            return ans;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(1)，因为只使用常数级额外变量。
