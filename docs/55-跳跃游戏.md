# [跳跃游戏](https://leetcode.cn/problems/jump-game/)

## 思路

贪心维护当前能到达的最远位置 `far`。
遍历下标 `i`：
- 若 `i > far`，说明无法到达当前位置，直接返回 false
- 否则更新 `far = max(far, i + nums[i])`
遍历结束即可以到达末尾。

时间复杂度：O(n)。
空间复杂度：O(1)。

## 代码

=== "Python"
    ```python
    from typing import List

    class Solution:
        def canJump(self, nums: List[int]) -> bool:
            far = 0
            for i, x in enumerate(nums):
                if i > far:
                    return False
                far = max(far, i + x)
            return True
    ```

=== "C++"
    ```cpp
    #include <vector>
    #include <algorithm>
    using namespace std;

    class Solution {
    public:
        bool canJump(vector<int>& nums) {
            int far = 0;
            for (int i = 0; i < (int)nums.size(); ++i) {
                if (i > far) return false;
                far = max(far, i + nums[i]);
            }
            return true;
        }
    };
    ```
