# [和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/)

## 思路

前缀和 + 哈希表。
设当前前缀和为 `pre`，若此前出现过前缀和 `pre-k` 共 `c` 次，则以当前位置结尾、和为 `k` 的子数组新增 `c` 个。
哈希表记录每个前缀和出现次数。

时间复杂度：O(n)。
空间复杂度：O(n)。

## 代码

=== "Python"
    ```python
    from typing import List
    from collections import defaultdict

    class Solution:
        def subarraySum(self, nums: List[int], k: int) -> int:
            cnt = defaultdict(int)
            cnt[0] = 1
            pre = 0
            ans = 0
            for x in nums:
                pre += x
                ans += cnt[pre - k]
                cnt[pre] += 1
            return ans
    ```

=== "C++"
    ```cpp
    #include <vector>
    #include <unordered_map>
    using namespace std;

    class Solution {
    public:
        int subarraySum(vector<int>& nums, int k) {
            unordered_map<int, int> cnt;
            cnt[0] = 1;
            int pre = 0, ans = 0;
            for (int x : nums) {
                pre += x;
                if (cnt.count(pre - k)) ans += cnt[pre - k];
                ++cnt[pre];
            }
            return ans;
        }
    };
    ```
