# [最短无序连续子数组](https://leetcode.cn/problems/shortest-unsorted-continuous-subarray/)

## 思路

一次从左到右记录当前最大值，若 `nums[i] < max_seen` 则右边界更新为 `i`。
一次从右到左记录当前最小值，若 `nums[i] > min_seen` 则左边界更新为 `i`。
若数组本来有序，返回 0。

## 代码

=== "Python3"
    ```python

    class Solution:
        def findUnsortedSubarray(self, nums: List[int]) -> int:
            n = len(nums)
            left, right = n, -1
            max_seen = -10**18
            min_seen = 10**18
            for i in range(n):
                max_seen = max(max_seen, nums[i])
                if nums[i] < max_seen:
                    right = i
            for i in range(n - 1, -1, -1):
                min_seen = min(min_seen, nums[i])
                if nums[i] > min_seen:
                    left = i
            return 0 if right == -1 else right - left + 1
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int findUnsortedSubarray(vector<int>& nums) {
            int n = (int)nums.size();
            int left = n, right = -1;
            int max_seen = INT_MIN, min_seen = INT_MAX;
            for (int i = 0; i < n; ++i) {
                max_seen = max(max_seen, nums[i]);
                if (nums[i] < max_seen) right = i;
            }
            for (int i = n - 1; i >= 0; --i) {
                min_seen = min(min_seen, nums[i]);
                if (nums[i] > min_seen) left = i;
            }
            return right == -1 ? 0 : right - left + 1;
        }
    };
    ```

## 复杂度分析

时间复杂度：O(n)。
空间复杂度：O(1)。
