# [62. 不同路径](https://leetcode.cn/problems/unique-paths/)

## 思路

`dp[i][j]` 表示到达格子 `(i,j)` 的路径数。
转移：`dp[i][j] = dp[i-1][j] + dp[i][j-1]`。
第一行和第一列都为 1。

时间复杂度：$O(mn)$。
空间复杂度：$O(mn)$，可优化到 $O(n)$。

## 代码

=== "Python"
    ```python
    class Solution:
        def uniquePaths(self, m: int, n: int) -> int:
            dp = [[1] * n for _ in range(m)]
            for i in range(1, m):
                for j in range(1, n):
                    dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
            return dp[m - 1][n - 1]
    ```

=== "C++"
    ```cpp
    #include <vector>
    using namespace std;

    class Solution {
    public:
        int uniquePaths(int m, int n) {
            vector<vector<int>> dp(m, vector<int>(n, 1));
            for (int i = 1; i < m; ++i) {
                for (int j = 1; j < n; ++j) {
                    dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
                }
            }
            return dp[m - 1][n - 1];
        }
    };
    ```
