# [回文子串](https://leetcode.cn/problems/palindromic-substrings/)

## 思路

中心扩展。每个位置作为奇数中心和偶数中心各扩展一次，
每扩展成功一次就累计一个回文子串。

## 代码

=== "Python3"
    ```python
    class Solution:
        def countSubstrings(self, s: str) -> int:
            n = len(s)
            ans = 0

            def expand(l: int, r: int) -> int:
                cnt = 0
                while l >= 0 and r < n and s[l] == s[r]:
                    cnt += 1
                    l -= 1
                    r += 1
                return cnt

            for i in range(n):
                ans += expand(i, i)
                ans += expand(i, i + 1)
            return ans
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int expand(const string& s, int l, int r) {
            int cnt = 0;
            while (l >= 0 && r < (int)s.size() && s[l] == s[r]) {
                ++cnt;
                --l;
                ++r;
            }
            return cnt;
        }

        int countSubstrings(string s) {
            int ans = 0;
            for (int i = 0; i < (int)s.size(); ++i) {
                ans += expand(s, i, i);
                ans += expand(s, i, i + 1);
            }
            return ans;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n²)，通常来自双重循环或两两比较。
空间复杂度：O(1)，因为只使用常数级额外变量。
