# [70. 爬楼梯](https://leetcode.cn/problems/climbing-stairs/)

## 思路

设 `f[i]` 为到达第 `i` 阶的方法数，则
$f[i] = f[i-1] + f[i-2]$。
滚动维护前两项即可。

时间复杂度：$O(n)$。
空间复杂度：$O(1)$。

## 代码

=== "Python"
    ```python
    class Solution:
        def climbStairs(self, n: int) -> int:
            if n <= 2:
                return n
            a, b = 1, 2
            for _ in range(3, n + 1):
                a, b = b, a + b
            return b
    ```

=== "C++"
    ```cpp
    class Solution {
    public:
        int climbStairs(int n) {
            if (n <= 2) return n;
            int a = 1, b = 2;
            for (int i = 3; i <= n; ++i) {
                int c = a + b;
                a = b;
                b = c;
            }
            return b;
        }
    };
    ```
