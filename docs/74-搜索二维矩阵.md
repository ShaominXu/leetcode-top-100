# [搜索二维矩阵](https://leetcode.cn/problems/search-a-2d-matrix/)

## 思路

矩阵每行有序且行首大于上一行行尾，可视为一维有序数组。
将下标 `mid` 映射到二维坐标：
row = mid // n, col = mid \% n，直接二分查找。

## 代码

=== "Python3"
    ```python

    class Solution:
        def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
            m, n = len(matrix), len(matrix[0])
            l, r = 0, m * n - 1
            while l <= r:
                mid = (l + r) // 2
                x = matrix[mid // n][mid % n]
                if x == target:
                    return True
                if x < target:
                    l = mid + 1
                else:
                    r = mid - 1
            return False
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        bool searchMatrix(vector<vector<int>>& matrix, int target) {
            int m = (int)matrix.size(), n = (int)matrix[0].size();
            int l = 0, r = m * n - 1;
            while (l <= r) {
                int mid = l + (r - l) / 2;
                int x = matrix[mid / n][mid % n];
                if (x == target) return true;
                if (x < target) l = mid + 1;
                else r = mid - 1;
            }
            return false;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(log(mn)，主要开销来自排序/二分/堆等对数级操作。
空间复杂度：O(1)，因为只使用常数级额外变量。
