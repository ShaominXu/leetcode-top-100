# [颜色分类](https://leetcode.cn/problems/sort-colors/)

## 思路

荷兰国旗问题。维护三个区域：

- `[0, p0)` 都是 0
- `[p0, i)` 都是 1
- `(p2, n-1]` 都是 2
遍历时根据当前值与边界交换并推进指针。

## 代码

=== "Python3"
    ```python

    class Solution:
        def sortColors(self, nums: List[int]) -> None:
            p0, i, p2 = 0, 0, len(nums) - 1
            while i <= p2:
                if nums[i] == 0:
                    nums[p0], nums[i] = nums[i], nums[p0]
                    p0 += 1
                    i += 1
                elif nums[i] == 2:
                    nums[p2], nums[i] = nums[i], nums[p2]
                    p2 -= 1
                else:
                    i += 1
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        void sortColors(vector<int>& nums) {
            int p0 = 0, i = 0, p2 = (int)nums.size() - 1;
            while (i <= p2) {
                if (nums[i] == 0) {
                    swap(nums[p0], nums[i]);
                    ++p0;
                    ++i;
                } else if (nums[i] == 2) {
                    swap(nums[p2], nums[i]);
                    --p2;
                } else {
                    ++i;
                }
            }
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(1)，因为只使用常数级额外变量。
