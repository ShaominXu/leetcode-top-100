# [76. 最小覆盖子串](https://leetcode.cn/problems/minimum-window-substring/)

## 思路

滑动窗口 + 计数。`need` 记录目标字符需求，窗口右扩满足条件后，
不断左缩以得到更短合法区间。记录最短区间起点和长度。

时间复杂度：$O(n)$。
空间复杂度：$O(|\Sigma|)$。

## 代码

=== "Python"
    ```python
    from collections import Counter

    class Solution:
        def minWindow(self, s: str, t: str) -> str:
            need = Counter(t)
            missing = len(t)
            left = 0
            best_len = float('inf')
            best_l = 0
            for right, ch in enumerate(s):
                if need[ch] > 0:
                    missing -= 1
                need[ch] -= 1
                while missing == 0:
                    if right - left + 1 < best_len:
                        best_len = right - left + 1
                        best_l = left
                    c = s[left]
                    need[c] += 1
                    if need[c] > 0:
                        missing += 1
                    left += 1
            return "" if best_len == float('inf') else s[best_l:best_l + best_len]
    ```

=== "C++"
    ```cpp
    #include <string>
    #include <vector>
    #include <climits>
    using namespace std;

    class Solution {
    public:
        string minWindow(string s, string t) {
            vector<int> need(128, 0);
            for (char c : t) ++need[c];
            int missing = (int)t.size();
            int left = 0, best_len = INT_MAX, best_l = 0;
            for (int right = 0; right < (int)s.size(); ++right) {
                char ch = s[right];
                if (need[ch] > 0) --missing;
                --need[ch];
                while (missing == 0) {
                    if (right - left + 1 < best_len) {
                        best_len = right - left + 1;
                        best_l = left;
                    }
                    char c = s[left++];
                    ++need[c];
                    if (need[c] > 0) ++missing;
                }
            }
            return best_len == INT_MAX ? "" : s.substr(best_l, best_len);
        }
    };
    ```
