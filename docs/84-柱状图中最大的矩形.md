# [柱状图中最大的矩形](https://leetcode.cn/problems/largest-rectangle-in-histogram/)

## 思路

单调递增栈，栈中存下标。遍历时若当前高度小于栈顶高度，则弹栈并计算以弹出高度为高的最大面积。
宽度由当前下标和新栈顶确定。末尾补一个 0 触发清栈。

## 代码

=== "Python3"
    ```python

    class Solution:
        def largestRectangleArea(self, heights: List[int]) -> int:
            hs = heights + [0]
            st = []
            ans = 0
            for i, h in enumerate(hs):
                while st and hs[st[-1]] > h:
                    cur = hs[st.pop()]
                    left = st[-1] if st else -1
                    width = i - left - 1
                    ans = max(ans, cur * width)
                st.append(i)
            return ans
    ```

=== "C++"
    ```cpp

    class Solution {
    public:
        int largestRectangleArea(vector<int>& heights) {
            vector<int> hs = heights;
            hs.push_back(0);
            stack<int> st;
            int ans = 0;
            for (int i = 0; i < (int)hs.size(); ++i) {
                while (!st.empty() && hs[st.top()] > hs[i]) {
                    int cur = hs[st.top()];
                    st.pop();
                    int left = st.empty() ? -1 : st.top();
                    int width = i - left - 1;
                    ans = max(ans, cur * width);
                }
                st.push(i);
            }
            return ans;
        }
    };
    ```

### 复杂度分析

时间复杂度：O(n)，因为主要流程对数据做了一次线性处理。
空间复杂度：O(n)，额外空间主要来自哈希表/队列/递归栈等辅助结构。
