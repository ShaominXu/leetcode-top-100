# 48. 旋转图像

- LeetCode：[旋转图像](https://leetcode.cn/problems/rotate-image/)
- 分类：数组 / 矩阵

## 题意

给定一个 `n x n` 的二维矩阵 `matrix`，请你将图像顺时针旋转 90 度。

要求必须在原地旋转，不能使用另一个矩阵。

## 思路（转置 + 每行反转）

顺时针旋转 90 度可以拆成两步原地操作：

1. 主对角线转置：`matrix[i][j]` 与 `matrix[j][i]` 交换。
2. 每一行左右反转。

这两步做完后，结果就是顺时针旋转 90 度。

时间复杂度 $O(n^2)$，空间复杂度 $O(1)$。

=== "Python"
    ```python
    from typing import List


    class Solution:
        def rotate(self, matrix: List[List[int]]) -> None:
            n = len(matrix)

            for i in range(n):
                for j in range(i + 1, n):
                    matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]

            for row in matrix:
                row.reverse()
    ```

=== "C++"
    ```cpp
    #include <vector>
    #include <algorithm>
    using namespace std;


    class Solution {
    public:
        void rotate(vector<vector<int>>& matrix) {
            int n = (int)matrix.size();

            for (int i = 0; i < n; ++i) {
                for (int j = i + 1; j < n; ++j) {
                    swap(matrix[i][j], matrix[j][i]);
                }
            }

            for (int i = 0; i < n; ++i) {
                reverse(matrix[i].begin(), matrix[i].end());
            }
        }
    };
    ```
